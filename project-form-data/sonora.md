# Sonora — Project Form Data

**Title:** Sonora

**Headline:** Interactive audiobooks with AI narration and branching stories.

**Release Date:** June - 2024

**Link:** https://sonora-d09e63.gitlab.io/

## Description
Sonora was one of my most technically challenging projects. Working with real-time audio generation, word-level timing, and branching narratives required me to dive deep into audio processing and state management.

Integrating with ElevenLabs and OpenAI APIs taught me a lot about working with external services and handling async operations efficiently. Building the audio player with timeline controls and subtitle synchronization was a great learning experience in media handling.

This project at EdgenAI allowed me to work on cutting-edge technology and push the boundaries of what I thought was possible with web technologies. The satisfaction of seeing users navigate through interactive stories with AI-generated voices was incredible.

## Achievements
- Built a library of 10+ branching interactive stories with multiple endings.
- Generated real-time AI narration via ElevenLabs and OpenAI with word-level timing and subtitles.
- Added custom voice creation/management with high-quality TTS and pre-recorded audio fallback.
- Built a responsive player with timeline, replay, auto-advance, and choice navigation.
