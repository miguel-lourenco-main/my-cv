# Job AI Hub — Project Form Data

**Title:** Job AI Hub

**Headline:** AI-augmented job hunting command center — pipeline, documents, and interview prep in one workspace.

**Release Date:** February - 2026

**Link:** https://v0-job-hunting-platform-two.vercel.app

## Description
Job AI Hub started as a v0 prototype and grew into a product I actually wanted to use during my own job search. The hardest part wasn't any single feature — it was keeping profile, company, job post, application, and artifact data connected so AI generation always had the right context.

Building the action-taking chat (40+ tools), the application workspace with five linked tabs, and the interview code execution pipeline pushed me across the full stack: Supabase tenancy, structured AI outputs, PDF/resume parsing, and a testing setup that spans unit tests through Playwright e2e and k6 load tests.

I'm treating it as an MVP with invite-only access while I iterate on the marketing site and core workflows. The BYOK model was a deliberate choice — I wanted powerful AI without locking users into bundled pricing.

## Achievements
- Built an application-centric Kanban pipeline linking cover letters, interviews, Q&A, resumes, and notes per role.
- Created an action-taking AI chat assistant with 40+ tools to create and update records across the app.
- Added a job-post inbox that scrapes listings from URLs and converts them into applications.
- Shipped resume analysis/editing, AI cover-letter generation with version history, and an interview engine with mock sessions and code execution.
- Implemented BYOK (OpenAI, Anthropic, Google) with full JSON export/import for data portability.
