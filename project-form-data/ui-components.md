# UI Components Playground — Project Form Data

**Title:** UI Components Playground

**Headline:** Component library and playground to browse, test, and copy React UI across themes.

**Release Date:** February - 2025

**Link:** https://ui-components-5218c2.gitlab.io/

## Description
While working on my parents snack-bar website, I started another website meant to be a project with a simple objective that I could use as a base to transform into a useful and stunning app, where I had more creative freedom and direction to explore ideas and further consolidate what I learned so far.

In the end, it turned into a component web app where users have access to a library of components and themes that are interlinked, meaning that all components can implement any given theme style.

## Achievements
- Built one set of components that render across 6+ themes (Minimal, Modern, Gradient, Rounded, Dark, Neon).
- Added live prop editing with instant preview and production-ready code generation.
- Provided copyable, type-safe snippets and a theme gallery for side-by-side comparison.
- Designed a mobile-friendly interface with resizable panels and consistent light/dark behavior.
