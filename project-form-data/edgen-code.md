## Description
I also worked on an application similar to the previous one (EdgenChat), but this one needed to run as a Visual Studio extension, which I had no experience in doing. After digging through the VS documentation and some open source projects on GitHub, I was able to somewhat understand how to code a working VS extension and produce a working app that fit our purpose.

This project was particularly challenging because I had to learn the VS Code extension architecture from scratch. Understanding how extensions work, how to create webviews, and how to integrate with the VS Code API was a steep learning curve. However, it was very rewarding to see the extension working within the IDE and providing value to developers.

## Achievements
- Built a native VS Code extension delivering AI chat and code assistance inside the IDE.
- Learned the VS Code extension architecture and Webview API from scratch.
- Integrated the extension with the local model inference system API.
- Designed the experience to fit naturally into developer workflows.
