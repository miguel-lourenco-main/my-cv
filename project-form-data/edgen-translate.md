# EdgenTranslate — Project Form Data

**Title:** EdgenTranslate

**Headline:** LLM-powered document translation application for DOCX and PDF files.

**Release Date:** December - 2024

**Link:** https://edgentranslate-41c1b4.gitlab.io

## Description
After the previous projects, I went on to work on an application that would use an LLM to translate documents in DOCX and PDF formats, from any language to any language. This presents a significant challenge when it comes to extracting and replacing the translated content, since different languages can have sentences significantly smaller or bigger. In our case, we focused on generating high quality documents for specific languages for which we needed the results to be especially good.

This project taught me a lot about document processing, text extraction, and the complexities of maintaining document formatting across different languages. Handling the varying sentence lengths and ensuring the translated content fits properly within the document structure was a unique challenge that required careful engineering.

## Achievements
- Built an LLM translation pipeline for DOCX and PDF documents across any language pair.
- Preserved original document structure and formatting during translation.
- Solved content extraction/replacement challenges caused by differing sentence lengths between languages.
- Optimized output quality for specific high-priority target languages.
